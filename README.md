# Beehive_Drones_Intern
Penugasan intern beehive drones berupa Website Responsive.
Aplikasi frontend ini dibuat menggunakan React-Tailwind Framework

## Instalasi
```
npm install
npm start
Server akan berjalan di localhost:3000
```
